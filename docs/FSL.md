# FSL

## 基本信息

[FSL](https://fsl.fmrib.ox.ac.uk/fsl/fslwiki/)可以用于处理结构像、功能像和弥散像磁共振数据。可以在苹果和Linux系统上使用。大多数的工具可以使用图形界面。
